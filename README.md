# 🌟 Frontend Web Development Projects

This repository contains three frontend web development projects built using **HTML, CSS, and JavaScript**. These projects demonstrate responsive UI design, DOM manipulation, Local Storage, and interactive web application development.

---

# 📂 Projects

## 1️⃣ Expense Tracker

A simple and responsive expense tracking application that helps users manage their income and expenses.

### Features
- ➕ Add income and expense transactions
- ✏️ Edit transactions
- ❌ Delete transactions
- 💰 Automatic balance calculation
- 📊 Income & Expense summary
- 🔍 Search transactions
- 📂 Filter by category
- 🌙 Dark Mode
- 💾 Local Storage support

**Technologies Used**
- HTML5
- CSS3
- JavaScript
- Local Storage API

---

## 2️⃣ Music Player

A modern and responsive web-based music player that allows users to upload, manage, and play their favorite songs with an interactive interface.

### Features
- 🎵 Upload and play unlimited songs
- ▶️ Play & Pause songs
- ⏮️ Previous track
- ⏭️ Next track
- 🔀 Shuffle mode
- 🔁 Repeat mode
- 📂 Dynamic playlist
- 📊 Progress bar with seek functionality
- ⏱️ Current time & song duration
- 🔊 Volume control
- 🎨 Album cover display
- 💾 Persistent song library using IndexedDB
- 🗑️ Delete songs from playlist
- 📱 Fully responsive design

**Technologies Used**
- HTML5 Audio API
- IndexedDB
- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome

---

## 3️⃣ Blog Website (BlogSphere)

A responsive blog website where users can browse blogs, search articles, read blog posts, like them, and leave comments.

### Features
- 🏠 Responsive Home Page
- 🔍 Search Blogs
- 📂 Filter by Category
- 📖 Blog Details Page
- ❤️ Like System
- 💬 Comment System
- 🌙 Dark Mode
- 💾 Local Storage support

**Technologies Used**
- HTML5
- CSS3
- JavaScript
- Local Storage API
